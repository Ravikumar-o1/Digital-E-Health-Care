Installation
1>Install Node, Angular , MongoDB , Deep Learning Libraries

2>Open e-health care  and run the commands on terminal or cmd 

3>for hospital folder run command " ng serve --open" and for the folder e-health run command " npm start "

4>cd e-health-care\e-health and e-health-care\hospital

5> browser will be open and  the website will show the home page.

6> The mongo will create the database .
